第 1 條
-------
本標準依規費法第十條規定訂定之。

第 2 條
-------
本標準適用範圍如下：  
一、醫事憑證換發、補發之證照費。  
二、醫事憑證副卡、備用卡之核發證照費。

第 3 條
-------
醫事憑證之收費標準如下：  
┌────────┬──────────┐  
│收費項目        │收費標準            │  
│                │ (新臺幣元／每份)   │  
├────────┼──────────┤  
│醫事憑證換發    │  二百七十五元      │  
├────────┼──────────┤  
│醫事憑證補發    │  二百七十五元      │  
├────────┼──────────┤  
│醫事憑證副卡    │  二百七十五元      │  
├────────┼──────────┤  
│醫事憑證備用卡  │  二百七十五元      │  
└────────┴──────────┘

第 4 條
-------
本標準自發布日施行。
